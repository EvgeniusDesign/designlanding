# Мобільний лендинг — інтенсив «Графічний дизайн, який приносить гроші»

Статичний HTML/CSS/JS лендинг, зверстаний 1:1 з Figma-макета (mobile-first, ширина 390px).

## Структура
```
index.html            — розмітка всіх секцій
assets/
  css/styles.css      — усі стилі (токени, секції, кнопки, glow, torn-paper)
  js/main.js          — таймер акції + підстановка посилання на оплату
  fonts/              — СЮДИ покласти шрифти (див. fonts/README.txt)
  img/                — всі зображення/іконки (експортовані з Figma)
reference/            — еталонний рендер макета для звірки
```

## Що треба зробити перед запуском
1. **Шрифти.** Поклади у `assets/fonts/`:
   - `druk-cyr-medium.(woff2|otf|ttf)` — Druk Cyr Medium (заголовки)
   - `rage-italic.(woff2|otf|ttf)` — Rage Italic LET Remake (скрипт-акценти)
   Без них заголовки показуються системним запасним шрифтом.
2. **Посилання на оплату.** У `assets/js/main.js` встав URL у `PAYMENT_URL`
   — усі кнопки (`.cta`) підхоплять його автоматично.
3. (Опційно) Таймер: `COUNTDOWN_MINUTES` у `main.js` (за замовч. 10 хв).

## Локальний перегляд
Відкрий `index.html` у браузері (найкраще звузити вікно до ~390px,
або DevTools → режим телефону). У VSCode зручно через розширення Live Preview.
